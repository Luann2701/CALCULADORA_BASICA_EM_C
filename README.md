# Calculadora Simples em C
Projeto desenvolvido para praticar lógica de programação e manipulação de loops e condicionais na linguagem C.

## 🛠️ Funcionalidades
* Operações básicas: Soma, Subtração, Multiplicação e Divisão.
* Opção de continuar cálculos utilizando o resultado anterior.
* Interface via terminal.

## 🚀 Como executar
1. Tenha um compilador de C instalado (como o GCC) ou GDB ONLINE.
2. Compile o arquivo: `gcc calculadora.c -o calculadora`
3. Execute: `./calculadora`
